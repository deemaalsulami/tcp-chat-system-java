# TCP Multithreaded Chat System

A distributed chat application built using Java, TCP sockets, and multithreading.

## Features
- Real-time communication between multiple clients
- TCP socket networking
- Multithreaded server
- Message broadcasting
- Activity logging
- Admin commands

## Technologies
- Java
- TCP Sockets
- Multithreading
- File I/O

## Files
- ChatServer.java
- ChatClient.java
- multiClient.java
